AuraDraw yo‘riqnomasi

Chizish, layer, selection, rasm import, text va export bo‘yicha universal qo‘llanma.

Asosiy boshqaruv
Brush / Pencil / MarkerErkin chizish. Shift + sichqoncha g‘ildiragi — qalinlikni o‘zgartirish.
OpacityAlt + g‘ildirak — shaffoflikni o‘zgartirish. O‘ngdagi vertikal range ham ishlaydi.
ZoomCtrl + g‘ildirak — canvasni yaqinlashtirish yoki uzoqlashtirish.
ShapesShakl chizing, keyin 4 burchak va 4 tomondan resize/rotate qiling.
Text TCanvasga bosing, Paint kabi text-box ochiladi. Ctrl+Enter — canvasga joylash.
Bucket / PickerBucket rang bilan to‘ldiradi, Picker canvasdan rang olib beradi.
Layer va selection
Yangi toza layerLayer+ tugmasi har safar bo‘sh, toza layer yaratadi. Oldingi layer nusxalanmaydi.
Layer nusxaCopy tugmasi active layerdan nusxa olish uchun. Nom avtomatik “copy” bo‘ladi.
Layer nomiLayer panelidagi qalam iconi orqali nomni o‘zgartiring.
Rectangle SelectTanlangan to‘rtburchak ichidagi chizma alohida selection bo‘ladi.
Free SelectErkin chizilgan hudud ichidagi rasm ajratiladi.
DeleteSelection bor bo‘lsa faqat tanlangan qism modal tasdiq bilan o‘chadi.
Rasm, crop va export
Import imageRasm import bo‘lsa, move/resize/rotate qilinadigan selection sifatida chiqadi.
CropCtrl + Shift + X — tanlangan selection bo‘yicha canvasni kesadi.
ExportPNG/JPG tugmalari orqali active layer/canvasni yuklab oling.
Save / LoadLocalStorage orqali loyihani saqlash va qayta yuklash mumkin.
PrintPrinter iconi active canvasni chop etishga tayyorlaydi.
Dark / LightOy/quyosh tugmasi orqali mavzuni almashtiring.
